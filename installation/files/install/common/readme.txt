#### -- DEUTSCHE RELEASENOTES --

Bekannte Probleme:

1. Die Performance ist suboptimal. Ggf. Zyklen mit Strg-F11 reduzieren, um Soundprobleme zu minimieren, oder mit Strg-F12 erhöhen für gegebenenfalls höhere Framerate.
2. Es stehen momentan nur Windows-Builds zur Verfügung.

So geht’s los:

1. Nach der Installation solltest du auf ein sich öffnendes Kommandozeilenfenster beim Spielstart achten. Dort werden die Proben und Würfelwürfe während des Spiels dargestellt.
2. Alle Ausgaben werden zusätzlich in einer Datei `log.txt` im Pfad `%appdata%\Riva Dosbox Logger\` geloggt.

Für weitere Informationen, Fragen oder Feedback wende dich bitte an den entsprechenden [Thread im Forum](https://www.crystals-dsa-foren.de/showthread.php?tid=6059).


#### -- ENGLISH VERSION NOTES --

Known Issues:

1. Performance is suboptimal. If necessary, reduce cycles with Ctrl-F11 to minimize sound issues, or increase them with Ctrl-F12 for potentially higher framerate.
2. Currently, only Windows Builds are available.

Getting Started:

1. After installation, look for a command-line window that should open upon game start. In it, the checks and dice rolls will be displayed during the game.
2. All outputs will also be logged in a file named `log.txt` in the `%appdata%\Riva Dosbox Logger\` path.

For further information, questions, or feedback, please refer to the respective [thread in the forum](https://www.crystals-dsa-foren.de/showthread.php?tid=6059). Please note that the forum is in German.
